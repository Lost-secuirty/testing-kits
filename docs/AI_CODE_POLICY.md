# AI-Assisted Code Review Policy

This repository treats AI-assisted code as untrusted until it is reviewed, tested, and tied back to deterministic evidence.

The rule is operational, not philosophical: AI output may be useful, but it is not accepted on style or plausibility alone.

## Scope

This policy applies to:

- code generated by an AI assistant;
- code edited or refactored by an AI assistant;
